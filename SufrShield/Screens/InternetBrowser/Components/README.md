# Компоненты браузера

Этот каталог содержит все компоненты для интернет-браузера, организованные по принципу декомпозиции.

## Структура компонентов

### Основные компоненты

- **`WebView.swift`** - Базовая обертка для WKWebView с поддержкой навигации
- **`WebViewContainer.swift`** - Контейнер, управляющий состояниями WebView
- **`InternetBrowserView.swift`** - Главный экран браузера, объединяющий все компоненты

### UI компоненты

- **`AddressBar.swift`** - Адресная строка с полем ввода и кнопкой обновления
- **`NavigationBar.swift`** - Панель навигации с кнопками назад/вперед/домой/поделиться
- **`LoadingOverlay.swift`** - Индикатор загрузки страницы
- **`ErrorOverlay.swift`** - Экран ошибки с возможностью повтора

### Утилиты

- **`BrowserComponents.swift`** - Общие стили, константы и утилиты

## Принципы архитектуры

1. **Декомпозиция** - каждый компонент отвечает за одну функцию
2. **Переиспользование** - компоненты можно использовать в других частях приложения
3. **Чистота кода** - каждый файл содержит только связанную логику
4. **Тестируемость** - компоненты легко тестировать изолированно

## Использование

```swift
// Простое использование WebView
WebViewContainer(url: URL(string: "https://example.com")!)

// Использование с кастомными компонентами
VStack {
    AddressBar(url: $url, onGo: loadURL, onRefresh: reload)
    NavigationBar(canGoBack: true, canGoForward: false, ...)
    WebViewContainer(url: currentURL)
}
```
